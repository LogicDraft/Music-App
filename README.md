# Music App
Play a Musical Instruments using keyboard

</center>

## How to play 

- Use the keyboard to play the instruments
- Each key plays a different instrument

## Use of Code

- This code uses the HTML5 Audio API to play audio files
- It uses the KeyboardEvent API to detect key presses
- It uses the MouseEvent API to detect mouse clicks
- It uses the Web Audio API to play audio files
